# EJS Blog Page

## Install Package
```
npm install
```

## Port and nodemon
Port is at 3000
```
nodemon app.js
```
## Install MongoDB Community Version
```
mongosh
use blogDB
```

Install [MongoDB Locally](https://www.mongodb.com/docs/manual/administration/install-community/).
